# User Management MERN

A full-stack User Management System built using the MERN stack. This application allows users to register, log in, and manage user data through a modern React frontend and a RESTful Node.js/Express backend.

## 🚀 Features

- User Registration
- User Login Authentication
- Add New Users
- View User Details
- Update User Information
- Delete Users
- REST API Integration
- Responsive User Interface
- Database Connectivity
- Error Handling & Validation

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js
- CORS

### Database
- MongoDB / MySQL

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/onkarkarande77/User-Management-MERN.git
```

### Move into Project

```bash
cd User-Management-MERN
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000

```

Start Backend Server:

```bash
npm start
```

or

```bash
nodemon server.js
```

---

## Frontend Setup

Open a new terminal:

```bash
cd frontend
npm install
npm start
```

Application will run on:

```text
http://localhost:5173
```

Backend API:

```text
http://localhost:5000
```

---

## API Endpoints

### Register User

```http
POST /api/register
```

### Login User

```http
POST /api/login
```

### Get All Users

```http
GET /api/users
```

### Get User By ID

```http
GET /api/users/:id
```

### Update User

```http
PUT /api/users/:id
```

### Delete User

```http
DELETE /api/users/:id
```
